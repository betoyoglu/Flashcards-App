# AI-Powered Flashcard App 

An interactive, AI-driven flashcard application built with **Modern Android Development** standards. This app allows users to seamlessly generate study decks on any topic using Google's Gemini API and master them 
through an intuitive, Tinder-style swipe interface.

## Key Features

* **AI Deck Generation:** Instantly create comprehensive flashcard decks using the **Gemini API**.
* **Secure Authentication:** User signup and login seamlessly handled by **Firebase Authentication**.
* **Interactive Learning:** Tap to flip cards and reveal answers.
  * Swipe right to mark as "Learned" or swipe left to "Review Again".
* **Dashboard & Stats:** Track your overall progress, total created decks, and cards that need a review.
* **Modern UI/UX:** Fully built with **Jetpack Compose** using Material Design 3 guidelines and smooth gradient backgrounds.

## 🛠️ Tech Stack & Architecture

This project is built from scratch using the recommended Android architecture and modern tech stack:

* **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (100% Kotlin)
* **Architecture:** MVVM (Model-View-ViewModel) + Clean Architecture principles
* **State Management:** Kotlin Coroutines & StateFlow
* **Dependency Injection:** [Dagger Hilt](https://dagger.dev/hilt/)
* **Navigation:** Jetpack Navigation Compose
* **Local Storage:** [Room Database](https://developer.android.com/training/data-storage/room)
* **Backend & Auth:** Firebase Auth & Cloud Firestore
* **AI Integration:** Google Gemini API SDK
* **Animations:** [Lottie Compose](https://github.com/airbnb/lottie-android), SwipeableCard library
